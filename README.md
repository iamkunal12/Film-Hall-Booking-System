# Film Hall Booking System

A desktop-based **Film Hall Booking System** developed using **C++**, **Qt Framework**, and **SQLite**.  
This project allows users to view movies, select seats, book tickets, and manage bookings efficiently.

## Features
- Movie Listing
- Show Time Management
- Seat Selection & Booking
- Ticket Booking Confirmation
- Admin Management Panel
- Database Storage using SQLite

## Technologies Used
- C++
- Qt Creator (Qt Widgets)
- SQLite Database

## Project Structure
- UI designed using Qt Designer
- Backend logic implemented in C++
- SQLite used for storing movies, seats, bookings and payments

## How to Run the Project
1. Open **Qt Creator**
2. Click **Open Project**
3. Select the '.pro' file
4. Configure the Kit (Desktop Qt)
5. Click **Build and Run**

## Database
- SQLite database is used
- Tables include:
  - Movies
  - Seats
  - Bookings
  - payment
